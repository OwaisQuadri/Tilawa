## Playback Setup Sheet — Estimated Time Display

### Estimated Playback Time Footer
- Open the Play sheet and verify "Estimated time: ~X" appears in the Playback section footer
- Change the range (From/To ayah) → estimate updates live
- Adjust speed slider (0.5× to 2.0×) → estimate changes inversely
- Increase ayah repeat count → estimate scales proportionally
- Increase range repeat count → estimate multiplies accordingly
- Change gap between ayaat → estimate increases
- Toggle sliding window mode → estimate recalculates with solo/connection/full-range phases
- Adjust sliding window parameters (A/B/C/D) → estimate updates
- Set a repeat count to ∞ → shows "∞" instead of a time
- Verify with both CDN reciters (cached and uncached files) and local recordings
