## CDN UI & Storage

- Reciters tab rows should show only name, riwayah label, and source icons (no progress bars or badges)
- Tap a CDN reciter — the CDN source row should display a coverage badge with cache size (e.g. "45/114 surahs · 1.2 GB")
- Start a surah download, return to the detail view — the CDN row should show a progress bar with status text
- After download completes, the badge should update with the new surah count and cache size
- Swipe-delete a CDN source — cached audio should be automatically cleaned up
- Settings → Storage section shows "Clear Cache (X MB)" — tap to clear all cached CDN audio
